## Description

### API for registering items with expiration date, JWT protected routes

#

## Running the application with compose

</br>

```bash
$ docker compose -f "../docker-compose.yaml" up -d --build
```

# 

## To run locally

### Installation

</br>

```bash
$ npm install
```

### Running the app

</br>

```bash
# watch mode
$ npm run start

# development
$ npm run start:d

# production mode
$ npm run start:prod
```

#

### Test

</br>

```bash
# unit tests
$ npm run test

# test with report Nyan
$ npm run test:r

# test production mode
$ npm run test:prod
```
